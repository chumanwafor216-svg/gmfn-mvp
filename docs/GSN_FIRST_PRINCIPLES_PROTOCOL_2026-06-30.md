## 1. Purpose

This protocol defines the non-negotiable principles upon which every future line
of GSN code must be built.

Every developer joining the project should read this protocol before reading
any API documentation.

This is the philosophy behind the architecture.

---

## 2. The First Principle

Communities already create trust.

GSN does not.

Every feature must reinforce this statement.

If a feature assumes GSN creates trust, it violates the architecture.

---

## 3. The Second Principle

Communities own governance.

GSN owns infrastructure.

Communities make decisions.

GSN records decisions.

Communities recognise members.

GSN preserves recognition.

Never reverse these responsibilities.

---

## 4. The Third Principle

Evidence comes before trust.

Trust is never entered.

Trust is never typed.

Trust is never assigned.

Behaviour is recorded.

Evidence is preserved.

Trust emerges.

---

## 5. The Fourth Principle

Nothing important disappears.

History is never overwritten.

History grows.

Corrections create new events.

Leadership changes.

Community Memory remains.

Members leave.

Trust history remains.

Every important action becomes part of permanent institutional memory.

---

## 6. The Fifth Principle

One platform.

Many Community Domains.

Never create:

- School Version;
- Church Version;
- Market Version;
- Hospital Version.

Build one platform.

Configure through Community Settings.

---

## 7. The Sixth Principle

The nearest responsible leader should always know the member.

Authority should become more personal as it moves downward.

Trust should become stronger as distance becomes smaller.

This is why governance exists.

---

## 8. The Seventh Principle

Membership is not trust.

Identity is not trust.

Verification is not trust.

Endorsement is not trust.

Behaviour creates evidence.

Evidence builds Trust Passport.

Trust Passport supports decisions.

---

## 9. The Eighth Principle

Everything meaningful becomes an event.

```text
Member joined
-> Trust Event
```

```text
Verification responded
-> Trust Event
```

```text
Leadership changed
-> Governance Event
```

```text
Merchant released
-> Commerce Event
```

Nothing significant happens without leaving evidence.

---

## 10. The Ninth Principle

Every relationship matters.

GSN records:

- People;
- Communities;
- Departments;
- Lines;
- Leadership;
- Endorsements;
- Verification;
- Trust Events.

The value of GSN is not only in individual records.

It is also in the relationships between them.

---

## 11. The Tenth Principle

Opportunity should follow trust.

Never create visibility without accountability.

Never create commerce without evidence.

Never create verification without provenance.

---

## 12. The Eleventh Principle

Protect simplicity.

Whenever two solutions exist, choose the one that:

- uses existing engines;
- uses Community Settings;
- preserves evidence;
- reduces user effort.

Complexity is a cost.

---

## 13. The Twelfth Principle

Every feature must answer one question:

> Which engine owns me?

If the answer is unclear, the architecture is incomplete.

---

## 14. The Thirteenth Principle

Every future feature must strengthen at least one of these:

- Community;
- Trust;
- Opportunity;
- Commerce;
- Knowledge;
- Infrastructure.

If it strengthens none of them, it probably does not belong inside GSN.

---

## 15. Founder's Rule

The purpose of GSN is not to build another application.

The purpose is to build a permanent Trust Infrastructure capable of serving
communities across generations.

Every architectural decision should therefore be judged by one question:

> Will this still make sense when GSN supports millions of members, thousands of
> Community Domains and multiple countries?

If the answer is yes, the architecture is probably moving in the right
direction.

---

## Constitutional Statement

GSN is not a social network.

It is not a bank.

It is not a marketplace.

It is not a verification company.

It is a Community Trust Infrastructure.

Communities create value.

Behaviour creates evidence.

Evidence creates trust.

Trust creates opportunity.

Opportunity creates prosperity.

Everything inside GSN exists to strengthen that cycle while preserving the
memory, dignity, and demonstrated value of the communities it serves.
