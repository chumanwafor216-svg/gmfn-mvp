# GSN 3D Icon Assets

This folder is reserved for the GSN premium realistic 3D icon family.

Target style:

```text
premium skeuomorphic / realistic 3D icons
```

Rules:
- Use transparent-background `webp` or `png` files for compact UI icons.
- Keep the same visual family, angle, lighting, and material feel.
- Use navy, gold, white, and controlled green.
- Do not place literal photos here as UI icons.
- Do not add flat, outline, faded, cartoon, or emoji-style primary icons.

See `docs/GSN_ICON_MIGRATION.md` before adding or replacing icon assets.
